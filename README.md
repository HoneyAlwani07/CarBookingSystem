# Car Booking System
A Java-based car booking application built as my OOP final project in 2nd semester.
# Features
- Admin and Customer login
- Book and manage cars
- Admin dashboard to manage users and cars
- MySQL database integration
# Tech Stack
- Java
- Java Swing (GUI)
- MySQL (Database)
- JDBC
# How to Run
1. Run `database_setup.sql` in MySQL
2. Open `DatabaseConnection.java` and enter your MySQL password
3. Run `Main.java`
